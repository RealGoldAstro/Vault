Leaderboard Tags API index

Leaderboard tags let you attach dynamic text to leaderboard entries, such as streaks or season ranks. These helper methods all live on TagAPI.

Leaderboard methods

TagAPI:GiveLeaderboardTag

Give a leaderboard tag with dynamic text.

TagAPI:UpdateLeaderboardTag

Update the dynamic text for an existing leaderboard tag.

TagAPI:RemoveLeaderboardTag

Remove a leaderboard tag from a player.

TagAPI:GetLeaderboardTagText

Read the current dynamic text for a leaderboard tag.
