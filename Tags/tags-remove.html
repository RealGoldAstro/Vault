TagAPI:Give

TagAPI:Give grants a tag to a player. If the player already owns the tag, the call is ignored and no duplicate is created.

Syntax


TagAPI:Give(player: Player, tagId: string)

Parameters

Parameter Type Description
player Player The player who should receive the tag.
tagId string The tag identifier defined in your CCTFF config.

Basic example


local ReplicatedStorage = game:GetService("ReplicatedStorage")
local CCTFF = require(ReplicatedStorage:WaitForChild("CCTFF"))

local TagAPI = CCTFF.TagAPI

game.Players.PlayerAdded:Connect(function(player)
    TagAPI:Give(player, "Dev")
end)

Giving multiple tags


local tagsToGive = { "Dev", "Tester", "VIP" }

game.Players.PlayerAdded:Connect(function(player)
    for _, tagId in ipairs(tagsToGive) do
        TagAPI:Give(player, tagId)
    end
end)
